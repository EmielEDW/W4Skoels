Een eenvoudige teksteditor is alles wat je nodig hebt om HTML te leren.

Leer HTML met een Teksteditor

Webpagina's kunnen worden gemaakt en gewijzigd met behulp van professionele HTML-editors.

Voor het leren van HTML raden we echter een eenvoudige teksteditor aan, zoals Notepad++

Het gebruik van een eenvoudige teksteditor is een goede manier om HTML te beginnen leren.

Volg de onderstaande stappen om uw eerste webpagina te maken met Notepad++

Stap 1: Maak de map

  1. Begin met een nieuwe map te maken.
  2. In deze map steek je een tekst file.
  3. Hernoem deze file naar "index.html".
  4. Right-Click op de file en selecteer bewerk met Notepad++.

Stap 2: Schrijf wat HTML code

Schrijf of kopieer de volgende HTML code in je teksteditor:

<!DOCTYPE html>
<html>
<body>

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

Let op: Vergeet niet om je document op te slaan! (ctrl + s)

Stap 3: Open de HTML file

Open het opgeslagen HTML document in een webbrowser naar keuze.
