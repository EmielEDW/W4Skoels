HTML Attributen

  • Alle HTML elementen kunnen attributen hebben
  • Attributen voorzien extra informatie over elementen
  • Attributen zijn altijd gespecificeerd in de start tag
  • Attributen komen gebruikelijk in paren zoals: name="waarde"

Het href Attribuut

De <a> tag defineert een hyperlink. Het <a> attribuut specifieerd een URL van de pagina waar de link naartoe gaat:

Voorbeeld

<a href="https://codepen.io/pen">Bezoek codepen</a>
Probeer het zelf!

Je zal meer leren over links in het HTML Links hoofdstuk.

Het scr Attriuut

De <img> tag word gebruikt om een image in de HTML pagina in te voegen. Het <src> attribuut specifieerd het pad naar de image zodat de computer weet waar de image staat in jouw mappenstructuur:

Voorbeeld

<img src="images/img_Emiel_De_Waele.jpg">
Probeer het zelf!

Er zijn 2 manieren om een URL te specifieren:

1. Absolute URL - Links naar een externe foto word genoteerd als: src="https://www.w4skoels/images/img_Emiel_De_Waele.jpg"

Pas op: Externe foto's kunnen copyright rechten hebben. Let er dus op dat als je een site publiceerd dat je het recht hebt om die foto op je site te hebben. Anders kan deze plots van je website gehaald worden.

2. Relative URL - Links naar een foto met een relatief pad worden genoteerd als: src="images/img_Emiel_De_Waele.jpg"

Tip: Het is bijna altijd het best om relative URL's te gebruiken. Deze zullen niet breken als je van domein veranderd.

Als je relative paden maakt let dan op deze dingen: Let op je mappenstructuur. Als je 1 naar boven wil gaan doe je: ../ en om direct naar de root folder te gaan de je: /

Het scr Attriuut

De <img> tag kan ook de attributen <width> & <heigth> bevatten, deze attributen specifieren de hoogte en breedte van de image:

Voorbeeld

<img src="images/img_Emiel_De_Waele.jpg" width="500" heigth="600">
Probeer het zelf!

Het alt Attribuut

Het verplichte <alt> attribuut voor de <img> tag specifieerd een alternative tekst voor een foto, als de foto voor enige reden niet laad word de alt tekst vertoond.

Voorbeeld

<img src="images/img_Emiel_De_Waele.jpg" alt="Een foto van Emiel">
Probeer het zelf!

Je zal meer leren over images in het HTML Images hoofdstuk.

Het style attribuut

Het <style> attribuut word normaal niet gebruikt. Alle styling zou in css moten gebeuren. Maar hiermee kun je in je HTML document zelf stylen:

Voorbeeld

<p style="color:red;">Dit is een rode paragraph.</p>
Probeer het zelf!

Je zal meer leren over styles in het HTML Styles hoofdstuk.

Het lang attribuut

Je zou altijd een <lang> attribuut moeten gebruiken bij je <html> tag, om de taal waarin het document is geschreven te verklaren. Dit is bedoeld voor de webbrowsers.

Het volgende voorbeeld toont hoe je het lang attribuut kan implementeren:

<!DOCTYPE html>
<html lang="en">
<body>
...
</body>
</html>

Land codes kunnen ook worden toegevoegd bij het lang attribuut. Dus de eerste 2 karakters defineren de taal en de laatste 2 het land.

Het volgende voorbeeld toont hoe je het lang element met de land code kunt gebruiken:

<!DOCTYPE html>
<html lang="en-US">
<body>
...
</body>
</html>

Het title Attribuut

Het <title> attribuut defineert extra informatie over een element.

De waarde van het title attribuut zal worden vertoond als een tooltip wanneer je met je muis over het element hovert:

Voorbeeld

<p title="Dit is een tooltip">Dit is een paragraph</p>
Probeer het zelf!

Hoofdstuk Samenvatting

  • Alle HTML elementen kunnen attributen hebben
  • Het href attribuut van <a> specifieerd de URL van de pagina waar de link naartoe gaat
  • Het src attribuut van <img> specifieerd het pad naar de foto
  • Het width en heighth attribuut van <img> zorgen voor de grootte van de foto
  • Het alt attribuut van <img> voorziet de foto van alternative tekst als de foto niet vertoond word
  • Het style attribuut word gebruikt om styles toe te voegen zoals kleur, grootte en meer
  • Het lang attribuut van de <html> tag declareert de taal van de webpagina
  • Het title attribuut defineert extra informatie over een element
