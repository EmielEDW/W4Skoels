HTML Elementen

❮ Vorige Volgende ❯

Een HTML element is gedefineert door een start tag, de content en de sluitingstag.

Wat is een HTML Element?

Een HTML element is gedefineert door een start tag, de content, en een sluitingstag:

<tagname>Content komt hier...</tagname>

Het HTML element is alles van de start tag tot de sluitingstag:

<h1>Mijn eerste heading</h1>

<p>Mijn eerste paragraph</p>

Start tag Element content End tag
<h1> Mijn eerste heading </h1>
<p> Mijn eerste paragraph </p>
<br> niks geen

Let op: Sommige HTML elementen hebben geen content (zoals het <br> element).
Deze elementen worden lege elementen genoemd. Lege elementen hebben geen sluitingstag!

Nested HTML elementen.

HTML elementen kunnen worden genest.

Alle HTML documenten bestaan uit geneste HTML elementen.

De volgende voorbeelden bestaan uit 4 elementen (<html>, <body>, <h1> en <p>):

Voorbeeld

<!DOCTYPE html>
<html>
<body>

<h1>Mijn eerste heading.</h1>
<p>Mijn eerste paragraph.</p>

</body>
</html>
Probeer het zelf!

Voorbeel uitgelegd

Het <html> element is het root element en het defineert het volledige HTML document.

het heeft een start tag <html> en een sluitingstag </html>.

Dan, in het <html> element is er een <body> element:

<body>

<h1>Mijn eerste heading.</h1>
<p>Mijn eerste paragraph.</p>

</body>

Het <body> element defineert het document zijn body.

Het heeft een start tag <body> en een sluitingstag </body>.

In het <body> element zitten 2 andere elementen: <h1> en <p>:

<h1>Mijn eerste heading.</h1>
<p>Mijn eerste paragraph.</p>

Het <h1> element defineert een heading

Het heeft een start tag <h1> en een sluitingstag </h1>:

<h1>Mijn eerste heading.</h1>

Het <p> element defineert een paragraph.

Het heeft een start tag <p> en een sluitingstag </p>:

<p>Mijn eerste paragraph.</p>

Sla nooit een sluitingstag over

Sommige HTML elementen zullen correct vertoond worden, zelf als je de sluitingstag vergeet:

Voorbeeld

<!DOCTYPE html>
<html>
<body>

<h1>Mijn eerste heading.
<p>Mijn eerste paragraph.

</body>
</html>
Probeer het zelf!

PAS OP: Vertrouw hier nooit op! Zorg altijd voor een sluitingstag wanneer nodig.

Lege HTML elementen

HTML elementen met geen content worden lege elementen genoemd.

De <br> tag defineert een line break, en is een leeg element zonder sluitingstag:

Voorbeeld

<p>Dit is een<br> paragraph met een line break.</p>
Probeer het zelf!

HTML is niet hoofdletter gevoelig

HTML tags zijn niet hoofdletter gevoelig: <P> betekend hetzelfde als <p>.

De HTML standaard vereist geen lowercase tags, maar iedereen gebruikt standaard lowercase. Dit raden we jou ook ZEKER aan om te doen.

Let op: Op school gebruiken we altijd lowercase
