HTML Documenten

Alle HTML documenten moeten starten met een document declaratie type: <!DOCTYPE html>.

Het HTML document zelf begint met <html> en sluit met </html>.

Het zichtbare deel van de HTML code staat tussen <body> en </body>.

De <!DOCTYPE> declaratie

De <!DOCTYPE> declaratie representeerd het document type, en helpt de browser om de webpagina correct te vertonen.

Het mag maar 1x in de code. Je zet dit helemaal van boven in de code voor de html tags.

De <!DOCTYPE> is niet hoofdletter gevoelig. Maar het word meestal geschreven in caps.

De <!DOCTYPE> declaratie voor HTML5 is:

<!DOCTYPE html>

HTML Headings

HTML Headings zijn gedefineert door de <h1> tot <h6> tags.

<h1> defineert de belangrijkste heading. <h6> defineert de minst belangrijke heading:

Voorbeeld

<h1>Dit is heading 1</h1>
<h2>Dit is heading 2</h2>
<h3>Dit is heading 3</h3>
Probeer het zelf!

HTML Paragraphs

HTML paragraphs zijn gedefineert door een <p> tag:

Voorbeeld

<p>Dit is paragraph 1</p>
<p>Dit is paragraph 2</p>
<p>Dit is paragraph 3</p>
Probeer het zelf!

HTML Links

HTML links zijn gedefineert door een <a> tag:

Voorbeeld

<a href="https://codepen.io/pen">Dit is een link</a>
Probeer het zelf!

De links hun bestemming word gespecifeerd in het href attribuut.

Attributen worden gebruikt om extra informatie te geven aan een html tag.

Je zal meer leren over attributen in een later hoofdstuk.

HTML Images

HTML links zijn gedefineert door een <img> tag.

De source file (src) & alternative tekst (alt) worden hier meegeven als attributen:

Voorbeeld

<img href="images/EmielDeWaele.jpg" alt="Dit is een foto van Emiel De Waele">
Probeer het zelf!

Hoe bekijk je HTML source code?

heb je ooit een webpagina gezien en je afgevraagd: "Hey! Hoe hebben ze dat gedaan?"

Bekijk HTML source code:

Rick-click in een HTML pagina en selecteer "Paginabron weergeven". Dit zal een nieuw venster openen met de volledige HTML source code.

Inspecteer een HTML element:

Right-click op een element of ergens anders op de pagina, kies dan "Inspecteer" om te zien uit welke elementen die html code bestaat. Je zal HTML & CSS code zien. Je kan ook in die inspecteer side panel de code in real time aanpassen. Dit zal later SUPER handig zijn voor wanneer je vlug iets wil testen in je eigen code bijvoorbeeld.
