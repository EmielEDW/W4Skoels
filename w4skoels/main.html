HTML is de standaard taal voor het maken van HTML pagina's.

Wat is HTML?

  • HTML staat voor Hyper Text Markup Language
  • HTML is de standaard taal voor het maken van HTML pagina's
  • HTML beschrijft de structuur van een webpagina
  • HTML bestaat uit een verzameling van elementen
  • HTML elementen vertellen de browser hoe ze de content moeten vertonen
  • HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

Een simpel HTML Document

Voorbeeld

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>This is a Heading</h1>
<p>This is a paragraph.</p>

</body>
</html>
Probeer het zelf!

Voorbeeld uitgelegd

  • Het <!DOCTYPE html> declaratie defineert dat het een HTML5 document is
  • Het <html> element is het root element van een HTML pagina
  • Het <head> element bevat meta informatie over het HTML document
  • Het <title> element specifieerd een titel voor de HTML pagina (wat getoont wordt in de browser's titel bar of in de pagina's tab)
  • Het <body> element defineert de body van het document, en is een container voor alle zichtbare content, zoals headings, paragraphs, images, hyperlinks, tables, lists, etc.
  • Het <h1> element defineert een grote heading
  • Het <p> element defineert paragraph

Wat is een HTML Element?

Een HTML element is gedefineert door een start tag, de content, en een sluitingstag:

<tagname>Content komt hier...</tagname>

Het HTML element is alles van de start tag tot de sluitingstag:

<h1>Mijn eerste heading</h1>

<p>Mijn eerste paragraph</p>

Start tag Element content End tag
<h1> Mijn eerste heading </h1>
<p> Mijn eerste paragraph </p>
<br> niks geen

Let op: Sommige HTML elementen hebben geen content (zoals het <br> element).
Deze elementen worden lege elementen genoemd. Lege elementen hebben geen sluitingstag!

Web Browser

Het doel van een webbrowser is het lezen van de HTML documenten en het op een juiste manier te vertonen.

Een browser vertoont de HTML tags niet, maar gebruikt ze om te bepalen hoe het document moet vertoond worden.

HTML pagina structuur

Hieronder is een visualisatie van een HTML pagina structuur:

<html>
<head>
<title> Page title </title>
</head>
<body>
<h1> Dit is een heading </h1>
<p> Dit is een paragraph </p>
<p> Dit is een andere paragraph </p>
</body>


</html>

Let op:De content in de <body> sectie (het witte gedeelte) wordt vertoond in de browser.
De content in het <title> element zal NIET gevisualiseerd worden op de pagina zelf.
